# 🧙 MemeGPT - Your Personal Meme Generator

Generate hilarious memes with ease using MemeGPT! This application leverages the power of AI to intelligently select meme templates and generate relevant captions.

## ✨ Features
- **AI-Powered**: Uses GPT-4o and Gemini models.
- **Imgflip Integration**: Access to 100+ popular templates.
- **Manual Editor**: Full control over text and styling.
- **Async Processing**: Fast, non-blocking generation with ARQ.
- **Cloud Native**: R2 storage and Docker support.

## 🛠️ Quick Start

### 1. Clone and Setup
```bash
git clone https://github.com/your-repo/MemeGPT.git
cd MemeGPT
```

### 2. Configure
Create a `.env` file in `backend/` based on `.env.example`.

### 3. Run with Docker
```bash
docker-compose up -d
```

For detailed manual setup instructions, see the **[Getting Started](./DOCUMENTATION.md#3-getting-started)** section in the main documentation.
