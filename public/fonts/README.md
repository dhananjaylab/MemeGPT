# Font Files Required

This directory should contain the following font files for proper meme generation:

- `impact.ttf` - Used for most meme templates
- `ComicSansMS3.ttf` - Used for UNO Draw 25 Cards meme
- `ARIAL.TTF` - Used for Left Exit 12 Off Ramp meme

## Installation Instructions

1. Download the required fonts from a legitimate source
2. Place them in this directory
3. Ensure the filenames match exactly as listed above

## Fallback Behavior

If fonts are missing, the system will fall back to default system fonts, but the appearance may not match the traditional meme style.
